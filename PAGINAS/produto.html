TIVIT

A overflowpropriedade especifica o que deve acontecer se o conteúdo transborda caixa de um elemento. Esta propriedade especifica se para cortar conteúdo ou adicionar barras de rolagem quando o conteúdo de um elemento é grande demais para caber em uma área especificada. Nota: A overflowpropriedade só funciona para elementos de bloco com uma altura especificada.
